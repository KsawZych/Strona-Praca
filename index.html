Możliwości formatowania tekstu w HTML


  • Pogrubienie: < b > < / b >
  • Kursywa: < i > < / i >
  • Podkreślenie: < u > < / u >
  • Przekreślenie < strike > < / strike >
  • Indeks górny < sup > < / sup >
  • Indeks dolny < sub > < / sub >
  • Przejście do następnego wiersza w tekście: < br >
  • Zapobieganie automatycznemu przenoszeniu fragmentu tekstu do następnego wiersza: < norb > < / norb >
  • Spacja niełamliwa: & nbsp

Po niżej są trzy numerowania z różnymi znakami numerowania

Ty zaproponuj swoje, związane z informatyką

Nagłówki w treści strony WWW

  1. < h1 > Będzie największym nagłówkiem
  2. < h2 > ... < h5 > Będą coraz mniejsze
  3. < h6 > Będzie najmniejszym nagłówkiem

Możliwości funkcji font

  1. < font color="kolor czcionki" >
  2. < font size="rozmiar czcionki" >
  3. < font face="rodzaj czcionki" >

Najprostrze rodzaje kolorów do użycia w funkcji font

  1. Nazwa: red - czerwony, green - zielony, blue - niebieski
  2. RGB: rgb(255,0,0) - czerwony, rgb(0,255,0) - zielony, rgb(0,0,255) - niebieski
  3. Heksadecymalny, #RRGGBB: #FF0000 - czerwony, #00FF00 - zielony, #0000FF - niebieski
